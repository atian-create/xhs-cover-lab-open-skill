# Sample Track Route

Input:

> 我想做本地生活账号，城市是南宁，主要发美食和周末去哪玩。

Output:

## 最适合的封面类型

美食 / 本地生活欲望封面。

## 备选类型

1. 旅游 / 城市攻略封面
2. 互动帖封面

## 为什么适合

本地生活需要让用户快速判断：

- 这是哪个城市
- 值不值得去
- 多少钱
- 在哪个区域
- 是否能解决我今天吃什么 / 周末去哪的问题

所以封面不能只拍得好看，还要有城市词、场景、价格或判断。

## 可以测试的封面标题

1. 南宁这家老友粉，真的有点东西
2. 周末来南宁，别只知道去青秀山
3. 把你们南宁私藏小店交出来

## 不建议

不建议一开始做电影感旅行封面。它看起来高级，但对素材、拍摄、剪辑和持续更新要求都更高。
