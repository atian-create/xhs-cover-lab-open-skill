# Track-To-Cover Router

Use this file when the user tells you their content track but has not chosen a
cover style.

## AI Tools / Agent / Workflow

Primary styles:

- AI Person Tool Infographic
- Text-Dense Screenshot Dry-Good Cover
- No-Person Knowledge Card

Advice:

- Show the job done by AI, not only the word "AI".
- Strong covers include tool name, time saved, result preview, "from 0 to 1",
  "no code", "template", or "full workflow".

Avoid:

- fake technical UI
- generic futuristic posters
- text so dense that the first page cannot be scanned

## Self-Media / Creator Operation / Money-Making

Primary styles:

- Face-Led Big-Keyword Cover
- Person Plus Proof Scene
- Text-Dense Screenshot Dry-Good Cover

Advice:

- Put the platform, method, result, or workflow keyword on the cover.
- Good keywords include: 自媒体, 小红书, 剪辑, 封面, 涨粉, 变现, SOP, 新手.

Avoid:

- plain selfies without proof
- vague slogans such as "努力就会成功"

## Female Growth / Life Narrative

Primary styles:

- Face-Led Big-Keyword Cover
- Room-As-Identity Lifestyle Cover
- Text-Dense Screenshot Dry-Good Cover
- No-Person Knowledge Card

Advice:

- Name the actual desire: freedom, money, starting over, living alone, career
  confidence, relationship clarity, not being trapped, or a new life
  possibility.

Avoid:

- copying unrepeatable emotional life events
- using "女性成长" as a vague label without a concrete scene or desire

## Local Life / Food / Travel

Primary styles:

- Food / Local-Life Desire Cover
- Travel / City Information Cover
- Interaction Prompt Cover

Advice:

- City, district, route, price, store, or place keyword must be visible.
- Cross-city examples can be learned and translated into the user's city.

Avoid:

- pretty scenery with no decision value
- food photos that do not create appetite or practical use

## Good-Product Sharing / Product

Primary styles:

- Good-Product / Product Scene Cover
- Product conversion card when the user has a paid product or offer
- No-Person Knowledge Card when teaching selection rules

Advice:

- Show product plus usage scene.
- Let users understand when and why the product is useful.

Avoid:

- polished but unrepeatable lifestyle scenes
- adding price, scarcity, or income claims not provided by the user

## Workplace / Career / Education

Primary styles:

- Text-Dense Screenshot Dry-Good Cover
- Face-Led Big-Keyword Cover
- No-Person Knowledge Card

Advice:

- Make the audience stage explicit: college student, new employee, 35+, manager,
  teacher, job seeker, freelancer.
- A career cover should name a stage, tool, deliverable, or risk.

Avoid:

- using 35+ anxiety for college-student content
- vague career slogans without a task or scenario

## Interaction / Community Prompt

Primary style:

- Interaction Prompt Cover

Advice:

- The question must feed the account's next 1-3 normal posts.
- Good prompts are specific enough that users want to answer.

Avoid:

- broad controversy
- unrelated engagement bait
- stale copied prompts
